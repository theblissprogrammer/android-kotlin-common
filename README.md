# android-kotlin-common
A list of common helpers for kotln

Gradle example:
   ```
   allprojects {
        repositories {
            jcenter()
            maven { url "https://jitpack.io" }
        }
   }
   
   dependencies {
        implementation 'com.github.theblissprogrammer:android-kotlin-common:1.0.0'
   }
   ```
